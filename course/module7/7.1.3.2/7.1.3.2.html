ACL Best Practices

Using ACLs requires attention to detail and great care. Mistakes can be costly in terms of downtime, troubleshooting efforts, and poor network service. Before configuring an ACL, basic planning is required. The figure presents guidelines that form the basis of an ACL best practices list.

Figure is a table listing A.C.L. best practices. Guidelines include basing A.C.L.s. on the security policy of an organization, preparing a description of goals for each A.C.L., using a text editor to create, edit, and save each A.C.L., and testing A.C.L.s. in a test environment.
