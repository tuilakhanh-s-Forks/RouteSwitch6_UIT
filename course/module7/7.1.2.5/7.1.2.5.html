Wildcard Mask Keyword Examples

Example 1 in the figure shows how to use the any keyword to substitute the IPv4 address 0.0.0.0 with a wildcard mask of 255.255.255.255.

Example 2 shows how to use the host keyword to substitute for the wildcard mask when identifying a single host.

Note: The syntax for configuring standard IPv4 ACLs is covered later in this chapter.

Figure shows two examples using the C.L.I. commands for creating an access list that will permit any address. The examples begin in global configuration mode. The first example lists the method using wildcard masks. The command is, access hyphen list 1 permit zero dot zero dot zero dot zero 255 dot 255 dot 255 dot 255. The next example uses the keyword any, and the command is, access hyphen list 1 permit any. The second example is an A.C.L. that specifies a particular host. The command is, access hyphen list 1 permit 192 dot 168 dot 10 dot 10 zero dot zero dot zero dot zero. The next command in example two uses the keyword host. The command is, access hyphen list 1 permit host 192 dot 168 dot 10 dot 10.
